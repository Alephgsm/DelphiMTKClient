# Security Policy

This public repository is a product showcase and does not contain the full commercial source code.

## Reporting Security Issues

If you believe you found a security issue related to DelphiMTKClient, contact privately:

https://t.me/GsmCoder

Please do not publish sensitive details publicly before contacting the maintainer.

## Sensitive Material

Do not open issues or pull requests that include:

- Private keys.
- Device credentials.
- Customer data.
- Payload binaries.
- Commercial source code snippets.
- Loader files or protected assets.
- Exploit implementation details that should remain private.

## Intended Use

DelphiMTKClient is intended for developers, research teams and authorized service software development.

Use the project only for lawful development, research, repair, maintenance and authorized service scenarios.

## Public Repository Scope

This repository may contain product documentation, feature descriptions, screenshots and buyer-facing material. It should not contain the full licensed source package or sensitive implementation assets.
